# 🏥 Medicare - Application Web de Gestion Médicale

## 📌 Description
**Medicare** est une application web moderne conçue pour digitaliser la gestion des services médicaux.  
Elle permet aux patients, médecins et administrateurs d’interagir efficacement via une plateforme intuitive, sécurisée et performante.

L’application facilite la prise de rendez-vous, la gestion des profils et le suivi des interactions médicales.

---

## 🎯 Objectifs
- Simplifier la prise de rendez-vous médicaux  
- Améliorer la communication patient–médecin  
- Centraliser les données médicales  
- Offrir une expérience utilisateur fluide  

---

## 🚀 Fonctionnalités

### 👤 Patient
- Inscription et authentification  
- Consultation des médecins  
- Prise de rendez-vous en ligne  
- Suivi des rendez-vous  

### 👨‍⚕️ Médecin
- Gestion du profil professionnel  
- Gestion des disponibilités  
- Consultation des rendez-vous  

### 🛠️ Administrateur
- Gestion des utilisateurs (patients / médecins)  
- Gestion des spécialités médicales  
- Supervision globale de la plateforme  

---

## 🧑‍💻 Technologies utilisées

- **Frontend** : Angular  
- **Backend** : Spring Boot  
- **Base de données** : MongoDB  
- **Architecture** : API REST
